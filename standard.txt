User32NET 项目规范 1.0版本

类命名规范：采用大驼峰命名法
示例：ClassName;

文档注释示例：
/// <summary>
/// 函数说明:
/// 函数原型:
/// </summary>
/// <param name="pm">参数说明</param>
/// <return>返回值说明</return>

类型转换
C#      =>  C/C++
IntPtr  => HWND, UNIT, char *, wchar_t *, void *, int *, short *, long *, long long *, float *, double *, HINSTANCE
byte    => char
char    => wchar_t
int     => int, DWORD
short   => short
long    => long, long long
float   => float
double  => double
bool    => BOOL
